# Инструкция по использованию проекта

Есть три основных шага, которые выполняются командами в терминале.

### Шаг 1: Создание данных (если нужно)

Этот шаг создает примеры аудиофайлов для обучения. Обычно его нужно выполнить только один раз.

```bash
python project/generate_data.py
```
Файлы появятся в папке `project/data`.

### Шаг 2: Обучение моделей

Эта команда берет данные из шага 1 и обучает на них две модели: одну для "настоящих" звуков, другую для "поддельных".

```bash
python project/train_gmm.py
```
Модели сохранятся в папку `project/models`.

### Шаг 3: Проверка нового аудиофайла

Это главная функция. Возьмите любой `.wav` файл и проверьте его с помощью этой команды.

```bash
python project/predict.py --file /путь/к/вашему/файлу.wav
```
**Важно:** Замените `/путь/к/вашему/файлу.wav` на реальный путь к вашему файлу.

Программа напишет, считает ли она этот файл "настоящим" (real) или "поддельным" (fake).

## Структура папок:

- `project/data`: Здесь лежат аудиоданные.
- `project/models`: Здесь хранятся обученные модели.
- `project/`: Здесь находятся все рабочие скрипты (`.py` файлы).
